## Comandos 

1. Git log: Este comando funciona para determinar cuales fueron los cambios que se hicieron entorno a los commits 
2. Git merge: Este funciona para juntar los cambios que se hacen entre una rama y otra.
3. Git pull: Es un merge pero que se hacer desde la raíz
		Aquí se hacen cambios desde la actualización
4. Git reset-hard: Se regresa a lo que tenia anteriormente
5. Git status: Muestra los cambios que estaban dentro de la branch commit.
6. Git init: Inicializar los proyectos
7. Git add. Para añadir los archivos que se quieren subir
