
## Significa que:


- una parte está en **pasado**
    
- y la otra en **presente (o resultado actual)**
    

 Se usan cuando **una acción pasada afecta el presente** o **una situación actual explica un resultado pasado**.


## Estructura:

**If + past perfect → would + base verb**

#### Ejemplo:

- **If I had studied, I would have a better job now.**


# TIPO 2: PRESENTE → RESULTADO EN EL PASADO

##  Estructura:

**If + past simple → would have + V3**

### Ejemplo:

- **If I were taller, I would have played basketball.**

Traducción:  

	Si fuera más alto, habría jugado basquetbol.


|         **Tipo**         | **If (causa)**  |    **Resultado**    |                  **Ejemplo**                  |
| :------------------: | :---------: | :-------------: | :---------------------------------------: |
| 🔴 Pasado → Presente |  had + V3   |  would + base   |   If I had studied, I would be rich now   |
| 🟡 Presente → Pasado | past simple | would have + V3 | If I were rich, I would have bought a car |


# EJEMPLOS CLAROS

### 🔴 Pasado → presente

- If she had taken the job, she would be in another country now.
    
- If we had left earlier, we wouldn’t be late now.
    

---

### 🟡 Presente → pasado

- If I were smarter, I would have solved the problem.
    
- If he didn’t live so far, he would have come yesterday.
