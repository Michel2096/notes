## ¿Qué es una base de datos?

Una **base de datos** es una colección organizada de información estructurada, almacenada y accedida electrónicamente.

Un **SGBD (Sistema Gestor de Bases de Datos)** es el software que nos permite crear, leer, actualizar y eliminar datos. Ejemplos: MySQL, PostgreSQL, SQL Server, Oracle.
