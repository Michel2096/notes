
Las bases de datos no relacionales, no utilizan tablas, usan formatos distintos. 

### Tipos principales:

**1. Documentos (JSON)**

```
{
  "nombre": "Miguel",
  "edad": 22,
  "hobbies": ["programar", "leer"]
}
```


**2. Clave-valor**

```
"user_1" → "Miguel"
```

✔ Súper rápido  
✔ Usado en caché (como sesiones)


**3. Columnas**  
✔ Similar a tablas pero más flexible  
✔ Usado en sistemas grandes


**4. Grafos**  
✔ Guarda relaciones (tipo redes sociales)  
✔ Ejemplo: amigos, conexiones


### ¿Por qué existen?

Porque las bases tradicionales:

- Son rígidas
- No escalan bien para millones de datos

Las NoSQL:

- Escalan mejor
- Son más rápidas en grandes volúmenes
- Aceptan datos sin estructura fija
