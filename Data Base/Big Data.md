
Big Data = MUCHÍSIMOS datos.

Pero no solo es cantidad, también incluye:

### 🔹 Las 5 V:

- **Volumen** → enormes cantidades
- **Velocidad** → datos en tiempo real
- **Variedad** → diferentes formatos
- **Veracidad** → calidad de datos
- **Valor** → utilidad


### Ejemplo real:

- TikTok, Facebook, Amazon
- Generan millones de datos por segundo
