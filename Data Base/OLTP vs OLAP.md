
**OLTP** (Online Transaction Processing) es la base de datos del día a día de un negocio. Registra ventas, pedidos, movimientos bancarios, reservaciones — operaciones rápidas, de pocos registros a la vez, que deben ser exactas. Ejemplo: el sistema que usa una caja registradora.

---

**OLAP** (Online Analytical Processing) es la base de datos pensada para el análisis. Aquí preguntas cosas como "¿cuánto vendimos por región en los últimos 3 años?", lo cual implica leer millones de filas. Está optimizada para eso. Los Data Warehouses como BigQuery, Snowflake o Redshift son bases OLAP.
