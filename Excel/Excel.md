## Que es excel?


Es como una hoja de papel cuadriculada, pero digital y mucho más poderosa.

### 🔹 ¿Para qué se usa?

Excel se utiliza para:

- 📊 Hacer cálculos (sumas, promedios, etc.)
- 📈 Crear gráficas
- 📋 Organizar información
- 📉 Analizar datos
- 💼 Usos profesionales (contabilidad, negocios, ingeniería, etc.)

 Ejemplo real:  
 
Una empresa puede usar Excel para llevar control de ventas, gastos o empleados.

## Celda (Cell)

Es cada cuadrito dentro de la hoja.

 Aquí es donde escribes datos.

 Ejemplo:

- Texto: "Juan"
- Número: 150
- Fórmula: =SUMA(A1:A5)


## ¿Cómo se identifica una celda?

Cada celda tiene una dirección:

 **Columna + Fila**

Ejemplos:

- A1 → columna A, fila 1
- B3 → columna B, fila 3
- C10 → columna C, fila 10
