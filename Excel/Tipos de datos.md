
Excel puede trabajar con diferentes tipos de información.

	- Texto
	- Número
	- Fecha
	- Hora
	- Fórmulas
	- Booleano (VERDADERO/FALSO)
	- Moneda
	- Porcentaje
	- Datos mixtos
	

1. **Texto:** Son palabras o letras ==Hola, Miguel, Ventas 2026==
2. **Numero:** Son valores numéricos ==10. 3.14, -50==
3. **Fecha:** Excel guarda fechas como nueros internamente  ==- 01/01/2026 - 15-03-2025== Sirve para hacer cálculos de tiempo, meses, días, años etc.
4. **Hora** : Excel las puede manejar como fracciones del día ==10:30 AM,  18:45==
5. **Formulas**: Son expresiones que hacen cálculos: Se empiezan con un =
	1.Ejemplos:
		- =A1+B1
		- =SUMA(A1:A5)
	
6. **Booleano**: Solo tienen dos valores: VERDADERO, FALSO
		=A1>10 → devuelve VERDADERO o FALSO
		
7. **Moneda** Son números con formato de dinero 
		 - $100
		- $1,500.50
 8. **Porcentaje**: 10%, 50%
 9. **Datos mixtos**: Combinan datos y números:  
		 - "Producto 1"
		- "A123"

#Resumen 


Este tipo de datos se pueden representar dentro de Excel en cada una de las celdas
