
Sistema tipo IMSS pero modular:

- 🔐 Autenticación → AWS Cognito
- 🖥️ Frontend → React + TypeScript
- ⚙️ Backend → Flask (API REST)
- 🐘 Base de datos → PostgreSQL
- 🐳 Todo en Docker
- 🔮 Futuro → Microservicio IA con MongoDB + Python

```
proyecto/
├── docker-compose.yml
├── frontend/
│   ├── Dockerfile
│   └── app/
├── backend/
│   ├── Dockerfile
│   └── app/
├── db/
│   └── init.sql
```
